Final Project Level 4 Course 2021

In order to complete the course i had a final project to present to the teachers. This project aims to provide a database containing information about the computer equipment available in the classrooms through a dynamic website created in Dreamweaver. It was developed using PHP and HTML, and the database was created in MySQL, and subsequently connected to the website through XAMPP. It is possible to perform inventory maintenance, such as adding and removing equipment, with administrator permission, and to report existing technical issues with user permission, with the purpose of notifying the school's IT team for later resolution.

Technologies Used: HTML,CSS,PHP,XAMPP,MySQL,Dreamweaver

Project Restaurant 2023

The First Project of the Course was to develop a restaurant menu, to improve my skills using different classes. In this project I had a menu where i could add and remove, dishes, drinks and clients. Next step was to order products from the menu and calculate it's price. I created a way to issue receipts from each order. The last step was to report the client with most number of orders, best and worst-selling dish and drink, and the highest-profit dish from the entire sale list.

Technologies Used: C++ (Classes)
